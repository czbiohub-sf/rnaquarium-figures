# Data manifest — Fig4 pipeline

Every file the pipeline reads or writes, whether it is included in this folder, and where to obtain
the ones that don't. See `README.md` for how to run the notebooks.

## Code (included)

| file | role |
|------|------|
| `01_curate.ipynb` | curate the fish-associated virus set from metatranscriptome taxonomy + curation tables |
| `02a_reduce_matrix.R` | reduce the full Salmon count matrix to curated contigs (heavy / batch) |
| `02b_counts.ipynb` | collapse to per-virus counts + join metadata (supplemental table) |
| `03_figures.ipynb` | Figure 4 heat-trees, dot plots, 4D tissue panel, supplemental bar panels |
| `04_portal.ipynb` | portal artifacts: interactive heat-tree JSON + curated TSV/fasta |

## Human-curated inputs (included — small, the curation record)

| file | rows | produced by | used by | description |
|------|------|-------------|---------|-------------|
| `cluster_curation.tsv` | 55 | hand-curated | 01 | accepted clusters → curated name → Category (defines the fish-associated set) |
| `rejected_clusters.tsv` | — | bootstrapped by 01, then hand-reviewed | 01 | clusters excluded as non-fish. **NOT yet in this folder** — generated on 01's first run; add it here for the record. |
| `rejected_taxnames.tsv` | 20 | hand-curated | 01 | off-target taxa removed from within accepted clusters |
| `contig_split_overrides.tsv` | 2 | hand-curated | 01 | specific contigs reassigned to a second curated virus |
| `rescue_sequences.tsv` | 4 | hand-preserved | 01 | sequences for 4 contigs dropped upstream as adapters |
| `fig4b_family_curation.tsv` | 27 | hand-curated | 03 | Category \| Family \| Virus labels for the Figure 4B table |

## Large inputs (NOT included — point `<DATA_DIR>` at a copy; see Notes)

| file | used by | description |
|------|---------|-------------|
| virus taxonomy table (`taxonomy_hits_viruses_clusters_notargetsnosequenceforsalmon_mostrecent.tsv`, per-contig NT/NR + clusters) | 01 | the processed virus taxonomy from the metatranscriptomic pipeline |
| virus taxonomy table *with sequences* (`taxonomy_hits_viruses_nonphage_withsequenceandclusters_mostrecent.tsv`) | 01 | same, carrying contig sequences (for the fasta) |
| full Salmon count matrix (`salmon_counts_matrix.tsv.gz`, ~74k × ~180k) | 02a | per-contig expression across all runs |
| run-level sample metadata table (`all_zf_dates_devstage_tissue_tech_curated.tsv`) | 02b | tissue / devstage / date / country / sequencing metadata |

## Notes

- `01` regenerates `rejected_clusters.tsv` on first run (bootstrap), then it is hand-reviewed;
  the reviewed copy should be committed here.
- The large inputs are not redistributed in this repository due to size; obtain them from the portal
  and unzip 2 taxonomy tables (*see README*) and set `<DATA_DIR>` in each notebook's CONFIG block.
