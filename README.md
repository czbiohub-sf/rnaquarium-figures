# RNAquarium

This repository contains scripts and notebooks for analyzing and documenting the
RNAquarium pipeline as run on zebrafish RNA-seq data from NCBI SRA in April
2024. (As well as supporting details from mosquito and bat runs)

 - Figure 1: [RNAquarium pipeline](https://github.com/czbiohub-sf/rnaquarium-figures/tree/main/Fig1)
 - Figure 2: [Virus/microbe discoveries](https://github.com/czbiohub-sf/rnaquarium-figures/tree/main/Fig2)
 - Figure 3: [Host gene expression resource (clustering and metadata)](https://github.com/czbiohub-sf/rnaquarium-figures/tree/main/Fig3)
 - Figure 4: [Integrated host/microbe transcript abundance across the RNAquarium](https://github.com/czbiohub-sf/rnaquarium-figures/tree/main/Fig4)
 - Figure 5: [Interpretable, shallow AI models built with RNAquarium](https://github.com/czbiohub-sf/rnaquarium-figures/tree/main/Fig5)

## Usage
```sh
git clone https://github.com/czbiohub-sf/rnaquarium-figures
cd rnaquarium-figures
```
