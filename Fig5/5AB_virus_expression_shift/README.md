# Per-gene expression shifts associated with viral infection across the zebrafish transcriptome

Code accompanying the manuscript figure on virus-associated host gene-expression
shifts (**Figure 5**). For each virus, bulk zebrafish RNA-seq samples from a large
expression compendium are partitioned into a **virus-positive** set (samples with
detectable viral transcripts) and a **virus-negative** set (all remaining samples).
For every gene, a robust **expression shift** is computed as the difference between
the median normalized log-expression `log2(TMM-CPM + prior count)` of the two sets —
equivalent to the log2 fold change of median expression — with 95% bootstrap
confidence intervals. Applied across all ~22,000 genes per virus, this yields a
genome-wide background distribution of shifts. The analysis produces a clustered
heatmap of a curated immune-gene panel across fish-associated viruses, with each
virus's genome-wide shift distribution overlaid as a mini-violin.

This notebook reproduces:

- `figures/median_shift_fishOnly=True.svg` — immune panel × **18 fish-associated viruses** (Figure 5B)
- `figures/median_shift_endogenous_nonfish.svg` — the same panel for the endogenous / non-fish viruses

## How to run

Launch Jupyter **from this directory** so the relative paths resolve, then open and
run `4.compute_virus_effect_on_expr_flip.ipynb` top to bottom:

```bash
cd virus_expression_shift
jupyter lab          # or: jupyter notebook
```

The notebook supports two modes, controlled by two flags near the top:

```python
load_effect_size_from_file   = True
load_effect_distro_from_file = True
```

**1. Load precomputed results (default — no large download).**
With the flags `True` and the count matrix absent, the notebook loads the bundled
tables in `data/output/` and regenerates both figures into `data/output/`. This is
fast and reproduces the figures in `figures/`. (The notebook auto-detects the missing
matrix and prints that it is running in this mode.)

**2. Full recompute from the count matrix.**
Set both flags to `False` and place `counts_tmm-norm.parquet` at
`data/counts_tmm-norm.parquet`. The notebook then recomputes the median shifts and
95% bootstrap CIs for the immune panel (`n_boot=400`, `seed=42`) and the genome-wide
null distribution for every gene and virus (parallelized; `n_workers=12`),
overwrites the tables in `data/output/`, and redraws the figures.

> Note: a full recompute uses the gene panel in `immune_genes_v20260311.csv`. If that
> panel has been revised since the bundled `data/output/` tables were generated, the
> set of genes in the recomputed heatmap can differ from the bundled tables. Genes that
> appear in the precomputed tables but are not in the current category file are drawn
> under an `Unlabeled` row category.

## Method summary

- **Expression shift.** For gene *g* and virus *v*,
  `shift(g, v) = median(log-expr of g in virus-positive) − median(log-expr of g in virus-negative)`,
  where log-expr is `log2(TMM-CPM + prior count)`. Because expression is already on the
  log2 scale, the median shift is a robust estimate of the log2 fold change of median
  expression. See `utils_effect_size.compute_median_shift`.
- **Confidence intervals.** Nonparametric bootstrap (400 resamples, fixed seed); 95% CI
  from the 2.5th/97.5th percentiles.
- **Virus selection.** Fish-associated viruses detected in >10 samples (18 after manual
  curation); endogenous/non-fish and insufficient-evidence sequences are shown separately.
- **Visualization.** Seaborn clustermap (Euclidean distance, average linkage) on a
  diverging scale centered at zero; per-virus genome-wide shift distribution overlaid as
  a mini-violin. See `utils_analyze_effect_size.plot_clustered_heatmap_with_ci_and_violin_and_symbols`.

A full Materials & Methods description accompanies the manuscript.
